# Project Dashboard

A beautiful, animated hub that links to three job‑hunting tools:

- **Resume Builder** – Build ATS‑optimized resumes and export as PDF.
- **Cold Email Writer** – AI‑generated personalised cold emails.
- **Job Scraper** – Scrape job listings from multiple platforms.

## How to Use
1. Open `index.html` in any modern browser.
2. Click on a project card or its “Launch App” button.
3. You will be redirected to the respective tool.

## Deployment
Deploy as a static site on Vercel:
1. Push this folder to a GitHub repository.
2. Import the repository on [vercel.com](https://vercel.com) (no build command needed).
3. Get a public URL and share it.
