# Project Dashboard

A beautiful, animated hub that links to three job‑hunting tools:

- **Resume Builder** – Build ATS‑optimized resumes and export as PDF.
- **Cold Email Writer** – AI‑generated personalised cold emails.
- **Job Scraper** – Scrape job listings from multiple platforms.

## How to Use
1. Open `https://project-dashboard-eta-eight.vercel.app/` in any modern browser.
